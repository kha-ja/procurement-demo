# Procurement Demo

HTML mockups for Mack, an email-based procurement agent for construction GCs.

## Files

| File | Purpose |
|------|---------|
| `index.html` | Switcher with phone/laptop toggle (defaults to phone) |
| `mack-mockup-phone.html` | Mobile-optimized demo |
| `mack-mockup-laptop.html` | Desktop 70/30 split layout |

## Editing

- **Copy**: Search/replace text in HTML
- **Agent name**: Replace "Mack" and "mack@procure.it"
- **Colors**: Edit CSS `:root` variables

See README.md for full demo scenario and vendor details.
